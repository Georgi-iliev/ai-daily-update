New API · OpenAI
GPT-5.4 Computer-Use API
GPT-5.4 introduces native, state-of-the-art computer-use capabilities directly into OpenAI's Chat Completions and Responses APIs. Developers can now give agents the ability to observe a screen, click UI elements, type, and execute multi-step workflows across desktop applications — without integrating third-party automation layers. Combined with the 1M-token context window, agents can plan, execute, and verify tasks across long operational horizons.
Why it matters: For the first time, a frontier general-purpose model ships with computer-use baked in, collapsing what previously required a separate toolchain (browser agents, RPA, OS-level automation) into a single API call. This dramatically lowers the barrier for teams building autonomous digital-worker pipelines.
